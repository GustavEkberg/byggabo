## Codebase Patterns

- **Schema location**: `lib/services/db/schema.ts`
- **Drizzle relations**: Use `defineRelations()` with RQB v2 API
- **ID generation**: `createId()` from `@paralleldrive/cuid2`
- **Money fields**: `decimal('amount', { precision: 10, scale: 2 })`
- **Enum columns**: `text('status', { enum: ['VALUE1', 'VALUE2'] })`
- **reference_repo exclusion**: Add to tsconfig.json exclude array, vitest.config.ts exclude array
- **Dialog component**: Created custom `components/ui/dialog.tsx` using Base UI `@base-ui/react/dialog`
- **Server actions pattern**: One action per file, use Schema validation, Match.value for error handling
